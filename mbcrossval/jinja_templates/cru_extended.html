Extended mass balance calibration for the global reference glaciers

Here you can find results for the cross-validation when the following model parameters are varied:

  • Precipitation Factor
  • Melt Temperature
  • Liquid Precipitation Temperature
  • Temperature Laps Rate
Each possible parameter combination (1215) is cross-validated and plotted with respect to one of the parameters. The box-whisker plots show the full range of results and the red dots indicates the results when using the proposed standard values.

All results for the global reference glaciers are calculated with the CRU climate data set.
